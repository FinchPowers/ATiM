=== Installation ===

Please see the CTRNet Documentation centre for information on how to install ATiM on your webserver.
CTRNet Doc Center: http://www.ctrnet.ca/mediawiki/index.php/ATiM_Installation_Guide

If you are installing ATiM on Windows Server running IIS additional changes will need to be made before the system
will work. More detail can be found here: http://www.ctrnet.ca/mediawiki/index.php/IIS_Configuration_Guide

It is highly recommended to run ATiM on an Apache webserver.
